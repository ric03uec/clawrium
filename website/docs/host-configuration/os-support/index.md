# OS Support

Clawrium supports various operating systems as target hosts for agent deployment.

## Supported Operating Systems

| OS | Version | Architecture | Status |
|----|---------|--------------|--------|
| [Ubuntu 24.04](/docs/host-configuration/os-support/ubuntu/24-04) | 24.04 LTS | x86_64 | Tested |
| [macOS](/docs/host-configuration/os-support/macos) | - | - | TBD |

## What Does "Supported" Mean?

A supported operating system has:

- **Tested Ansible playbooks** - Installation, configuration, and management
- **Documented dependencies** - System packages required for agents
- **Verified compatibility** - Agent types confirmed to work

## Adding OS Support

Want support for a different operating system? See [Contributing](/docs/contributing) or open an issue on GitHub.
